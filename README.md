# Wordle Full-Stack Web Application

A modern full-stack Wordle game developed with React, Node.js, Express.js, and MongoDB Atlas.

## Live Demo

Frontend:
https://wordle-frontend-qmfx.onrender.com

## Features

- Player profile system
- Cloud-based player statistics
- Achievement system
- Daily streak tracking
- Best score tracking
- Hint system
- Virtual keyboard
- Responsive design
- MongoDB database integration
- Frontend and backend deployment with Render

## Technologies Used

- React
- Vite
- Node.js
- Express.js
- MongoDB Atlas
- Render
- Git
- GitHub

## How It Works

1. The player enters a username.
2. If the profile does not exist, it is created automatically.
3. Existing players continue with their saved statistics.
4. Game results are stored in MongoDB.
5. Statistics and achievements are synchronized across devices.
